我整理了一些常用的 SMTP 服务信息，方便查询。

## Office 365 邮箱:
| 项目 | 信息 |
| ------------ | ------------ |
| SMTP server name | smtp.office365.com |
| SMTP port | 587(TLS) / 25 |
| TLS/StartTLS | Enabled |
| Username/email address | 您的 Office 365 邮箱地址 |
| Password | 您的 Office 365 密码 |
